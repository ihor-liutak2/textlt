textlt Help

File
  Ctrl+O        Open a file
  Ctrl+S        Save the current file
  Ctrl+Q        Exit the editor

Navigation
  Arrow Keys    Move the cursor
  Home          Move to the start of the current line
  End           Move to the end of the current line
  Page Up       Move one page up
  Page Down     Move one page down
  Ctrl+Left     Jump to the previous word
  Ctrl+Right    Jump to the next word
  Ctrl+Up       Jump to the previous paragraph
  Ctrl+Down     Jump to the next paragraph
  Shift+Ctrl+Up Select to the previous paragraph
  Shift+Ctrl+Down Select to the next paragraph

Search & Replace
  Ctrl+F        Open Find Panel
  Ctrl+R        Open Replace Panel
  F3            Find Next Match
  Shift+Enter   Find Previous Match
  Escape        Close Find/Replace Panel

Editing
  Ctrl + /      Toggle Comment
  Ctrl + T      Toggle Case (Invert Uppercase/Lowercase for selection or current word)
  Tab           Indent selected lines, or insert spaces when nothing is selected
  Shift+Tab     Remove one indent level from selected lines or the current line
  Alt+Backspace  Delete the previous word
  Ctrl+Delete   Delete the next word
  Edit Menu     Convert Indents: 4 -> 2 Spaces
  Edit Menu     Convert Indents: 2 -> 4 Spaces

Selection & Clipboard
  Shift+Arrows  Extend the active text selection
  Mouse Drag    Select text inside the editor viewport
  Ctrl+A        Select all text in the current buffer
  Ctrl+C        Copy the active selection, or current line when nothing is selected
  Ctrl+X        Cut the active selection, or current line when nothing is selected
  Ctrl+V        Paste clipboard text into the editor
  Ctrl+Shift+V  Paste clipboard text when the terminal reserves Ctrl+V

Cloud TTS
  Ctrl+J        Read Text from Cursor (TTS Stage 1)

Menus
  Tab           Switch focus from File Explorer to Editor
  F1            Open this help window
  F2            Open the File menu
  F3            Open the Edit menu
  F4            Open the Options menu
  Escape        Close the active dialog or menu

Options
  Toggle Line Numbers changes the editor gutter immediately.
  Toggle File Explorer shows or hides the sidebar.
  Auto Pairing inserts matching brackets and quotes while typing.
  Smart Auto-Indent carries indentation onto new lines.
  Theme opens the dynamic theme selector.
